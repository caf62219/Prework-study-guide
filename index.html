HTML

  • Ul Ol and Li refer to different lists.

CSS

  • CSS is the style of the webpage.
  • Div: is the selector, then there is a declaration block with a property and a value.
  • Padding adds space around content inside an element.
  • * refers to all the elements on the page.
  • DRY: do not repeat yourself it refers to the process of reusing a sharing code to reduce the number of lines
  • . before card designates a class sector
